# LitGUI

A beautiful, audiophile-grade alternative UI for Volumio. LitGUI provides a polished playback, browse, and queue experience with high-quality album art, audio quality badges, and deep links into Volumio system pages.

It runs alongside Volumio's default UI — access it at `http://your-volumio:7777`, or set it as your default interface in Settings.

## Install

From the plugin store (Settings → Plugins), or manually:

    curl -sSL https://litgui.com/install.sh | bash

After install, enable LitGUI in Settings → Plugins → Installed, then open `http://your-volumio:7777`.

## Requirements

Volumio 3 or 4.
